🚀  HOW TO USE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Open  pixel-wind-portal.html  in any web browser
     (Chrome, Firefox, Edge — works OFFLINE, no server needed)
  2. Admin tab is selected by default
  3. Login with the admin credentials above
  4. Start adding students from the Students section

🎓  ADDING STUDENTS (Admin Steps)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Login as Admin
  2. Go to Students → click "+ Add Student"
  3. Fill in: Name, Email, Password, Phone, Course, Batch,
             Join Date, Total Fee
  4. Share the email + password you created with the student
  5. Student logs in using "Student" tab on login screen
